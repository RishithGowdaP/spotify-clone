🎵 Spotify Clone

A modern Spotify-inspired music player built using HTML, CSS, and Vanilla JavaScript.

---

✨ Features

✅ Play / Pause songs
✅ Next & Previous controls
✅ Interactive progress bar
✅ Volume control
✅ Dynamic song rendering
✅ Search functionality
✅ Responsive Spotify-style UI
✅ Keyboard shortcuts
✅ Local storage support

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
